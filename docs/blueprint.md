# **App Name**: Free Fonts by Tringledo

## Core Features:

- Automatic Font Catalog Generation: A build-time process to scan the '/public/fonts' directory, parse font metadata (e.g., name, designer, licensing), and generate a static JSON manifest for efficient client-side rendering without a dedicated backend.
- Interactive Font Gallery: Dynamically display all discovered fonts, each with its official name, a styled live text preview, and metadata for easy browsing and selection.
- Customizable Font Preview: An intuitive input field that allows users to type custom text and instantly see it rendered across all available fonts simultaneously, facilitating quick comparison and selection.
- One-Click Font Download: Prominently featured download buttons for each font entry, enabling users to easily download individual font files or a curated ZIP archive.
- Non-Commercial Use Clarification: A dedicated and clearly visible section explaining the non-commercial usage terms or licensing information for all fonts provided on the platform.
- Responsive Design for Designers: An adaptable and modern UI layout that ensures an optimal and seamless experience across a range of devices and screen sizes, crafted with a designer's workflow in mind.
- AI-powered Slogan Suggestions: A generative AI tool that suggests creative and varied phrases or short slogans. Users can quickly apply these suggestions to the font preview field to assess how fonts render different types of content.

## Style Guidelines:

- Primary interactive elements: A vibrant lime green, #A7D707, for buttons, links, and active elements.
- Background color: A deep muted green-grey, #151711, to create a refined canvas that lets typography stand out.
- Accent color: A bright, clear yellow, #F2DB57, used sparingly for highlights and notices.
- Headline font: 'Space Grotesk' for a bold, modern feel.
- Body font: 'Inter' for smooth readability.
- A spacious, grid-based, and typography-focused layout.
- Minimalist, crisp line icons.
- Subtle micro-interactions that create a fluid, polished user experience.
